About SmartLabel

SmartLabel is an AI-powered web application that helps consumers make informed decisions about the food products they consume. By analyzing ingredient labels, we provide instant health ratings and recommendations.

How It Works

  1. Image Upload: Users upload a photo of a food product's ingredient label
  2. OCR Processing: We use Tesseract OCR to extract text from the image
  3. Ingredient Analysis: Our system analyzes the extracted ingredients against a database of known unhealthy and processed ingredients
  4. Health Rating: Based on the analysis, we generate a health rating:
    • Healthy - Contains minimal concerning ingredients
    • Moderate - Contains some processed or concerning ingredients
    • Avoid - Contains multiple unhealthy ingredients

Technology Stack

  • Flask: Python web framework for the backend
  • Tesseract OCR: Open-source optical character recognition engine
  • Pillow: Python Imaging Library for image processing
  • HTML/CSS: Frontend user interface

Disclaimer

This application is designed to provide general guidance on food ingredients. It should not replace professional medical or nutritional advice. The analysis is based on commonly recognized unhealthy ingredients and may not be comprehensive. Always consult with healthcare professionals for personalized dietary recommendations.
